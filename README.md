# 📰 News App

A React-based News App that fetches real-time news using News API.

## 🚀 Features
- Latest news articles
- Category-based filtering
- Responsive UI
- API integration

## 🛠️ Tech Stack
- React.js
- Vite
- Bootstrap

## 🔑 Setup Instructions

1. Clone the repo:
   git clone https:(https://github.com/Aman113Prajapati)

2. Install dependencies:
   npm install

3. Create .env file:
   VITE_NEWS_API_KEY=your_api_key

4. Run project:
   npm run dev

## 📦 Build
npm run build
